üìã Instructions

Status Update:

  • Select the seller whose orders you want to update
  • Choose whether to update by Tracking IDs or Order Reference Numbers
  • Select the new status (Delivered, Return, Pending, or Cancel)
  • Enter the IDs/numbers either by typing or uploading a file
  • Click "Update Status" to process the bulk update

Tracking IDs Upload:

  • Select the seller
  • Download the template Excel file
  • Fill in Order Number and Tracking ID columns
  • Upload the file to update tracking IDs for those orders

Bulk Return Scan:

  • Enter or paste multiple tracking IDs (one per line or comma separated)
  • Or upload a file with tracking IDs
  • Click "Mark as Return" to update all matching orders to return status
  • Results will show how many orders were updated
